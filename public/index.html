Nova transcrição

Suba um áudio e gere legendas precisas com timestamp por palavra.

Pré-visualizar conteúdo 

                  

                

              

            


            

              Modo ideal para áudios gerados por TTS. Você já tem o texto, então o WhisperX só sincroniza os timestamps com altíssima precisão.
            

          


          
          

            
            
          


          
          

            

            

              
              

                
                

                  
                  
                

                

                  Modo CPU é compatível com qualquer máquina, mas processa mais devagar (5-15 min para 10 min de áudio).
                

                

                  Modo GPU é muito mais rápido (30s-1min para 10 min de áudio), mas exige placa NVIDIA com CUDA instalado.
                

                

                  Nenhuma GPU NVIDIA compatível detectada. Apenas CPU está disponível.
                

              


              
              

                
                
              


              
              

                
                
                

                  CPU não suporta float16. Use int8 (rápido) ou float32 (preciso).
